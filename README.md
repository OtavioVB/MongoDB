# MongoDB (NoSQL)

O [MongoDB](https://www.mongodb.com/docs/) é um [banco de dados orientado à documentos](https://aws.amazon.com/pt/nosql/document/), ao contrário de [banco de dados SQL](https://pt.wikipedia.org/wiki/SQL), ele possui uma flexibilidade quanto aos atributos e propriedades que compõem um valor, já que seus dados são armazenados em [JSON](https://aws.amazon.com/pt/documentdb/what-is-json/).
